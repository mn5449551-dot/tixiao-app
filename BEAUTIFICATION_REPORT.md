## 🎯 设计原则

### 1. 视觉层次
- **主要信息**：大字号、深色、粗体
- **次要信息**：小字号、浅色、常规体
- **状态信息**：品牌色或功能色突出
- **交互元素**：明确的 hover 和 active 状态

### 2. 色彩系统
- **品牌色**：#f26e24（橙色）用于主要操作和状态
- **辅助色**：紫色用于特殊功能（参考图模式）
- **中性色**：大地色系保持温暖感
- **渐变**：品牌色渐变用于重要元素

### 3. 间距和圆角
- **圆角**：xl (12px), 2xl (16px), 3xl (24px)
- **间距**：基于 4px 网格系统
- **阴影**：多层次阴影营造深度

### 4. 动效设计
- **入场**：fadeIn (200ms)
- **悬停**：200ms 过渡
- **面板切换**：300ms 过渡
- **脉冲**：用于状态指示

## 🚀 技术实现

### CSS 技术
- **渐变**：`bg-gradient-to-br/from-[var(--brand-500)] to-[var(--brand-600)]`
- **毛玻璃**：`bg-white/60 backdrop-blur-sm`
- **阴影**：`shadow-sm`, `shadow-md`, `shadow-lg`
- **圆角**：`rounded-xl`, `rounded-2xl`, `rounded-3xl`
- **过渡**：`transition-all duration-200`

### React 技术
- **条件渲染**：根据状态显示不同 UI
- **动画**：CSS animations + transitions
- **事件处理**：hover, click, focus 状态
