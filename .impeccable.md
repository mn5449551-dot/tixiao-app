# Design Context

## Users
洋葱学园的运营同学，需要在工作中快速生成图文营销素材。他们通常在工作场景下使用，需要高效、直观的工具来管理从需求到成品的完整流程。

## Brand Personality
- **专业可靠**：清晰的信息架构，明确的操作指引
- **温暖亲和**：大地色系，柔和的圆角，友好的交互
- **智能高效**：AI 辅助，自动化流程，现代化设计

## Aesthetic Direction
- **视觉风格**：现代简约，温暖大地色系，大圆角设计
- **设计语言**：卡片式布局，渐变效果，毛玻璃质感
- **动效设计**：流畅自然，快速响应，微妙的过渡动画
- **主题**：Light mode only，温暖明亮的视觉体验

## Design Principles
1. **清晰的信息层级**：通过字号、颜色、间距建立明确的视觉层次
2. **一致的交互反馈**：hover、active、focus 状态统一且明显
3. **现代化的视觉语言**：渐变、圆角、阴影营造深度和质感
4. **高效的导航体验**：直观的树状结构，清晰的流程指引
5. **专业的 AI 助手**：拟人化的对话界面，友好的交互体验

## Key Visual Elements

### Color System
- **Primary**: #f26e24 (温暖橙色)
- **Background**: #f7f3ee (米白色)
- **Surface**: #f8f3ef, #efe6df (浅米色系列)
- **Text**: #1d1714 - #b9ada4 (深棕到浅灰棕)

### Typography
- **Font**: Noto Sans SC + IBM Plex Mono
- **Scale**: 10px, 11px, 13px, 14px, 16px, 18px, 24px, 32px, 48px
- **Weights**: 400 (regular), 500 (medium), 600 (semibold), 700 (bold)

### Spacing
- **Base**: 4px grid system
- **Common**: 8px, 12px, 16px, 24px, 32px, 48px

### Components
- **Border Radius**: xl (12px), 2xl (16px), 3xl (24px)
- **Shadows**: card, panel, inset
- **Transitions**: 150ms (fast), 200ms (normal), 300ms (slow)

### Layout
- **Project Tree**: 260px width, gradient background
- **Canvas**: Flexible center area
- **Agent Panel**: 360px width, gradient background
- **Cards**: 380px width, rounded-3xl
